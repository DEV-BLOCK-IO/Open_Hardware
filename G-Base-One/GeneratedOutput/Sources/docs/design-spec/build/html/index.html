PREAMBLE

EH-Türwächter nextGen

This document specifies the hardware design of the EH-Türwächter nextGen Revision 0.0.0. It includes the detailed description of the different schematic sections, their intention and usage, as well as hints for the integration and interaction with other corresponding assemblies. At appropriate points, requirements and recommendations for the BSP development will be given.
